Клонирую репозиторий к себе на комп через GitHub Desktop.
Перемещаю созданные мной файлы в папку локального репозитория (который склонировала к себе на комп).
Перехожу в GitHub Desktop.
Во вкладке текущщего репозитория выбираю название созданного клиентом репозитория (git@example.com:example/test.git).
Во вкладке текущая ветка выбаю ветку master.
На этот момент должны отобразится изменения в репозитории(тоесть перемещенные мною файлы).
Ввожу название комита.
Нажимаю кнопку создать комит "Commit to..."
Пушу комит на GitHub, нажимая кнопку "Push origin".
